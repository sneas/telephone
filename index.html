Mobile phone mockup as a Web Component.

Wrap any HTML/CSS/JS code with the
<iphone-16-max></iphone-16-max> or
<pixel-9-pro></pixel-9-pro>
and it will be rendered inside a mobile phone frame.

Use this free
Web Component
to showcase a working iOS or Android app on your website.
